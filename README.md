# we-hires
Easy solutions for all your hiring problems


#How to use it
# 1. first clone it to local 
git clone https://github.com/aditya28vishwakarma-ai/we-hires.git

# 2. Pull down your brilliant new backend structure and frontend files
git pull origin main

# 3. Re-activate or set up their own local virtual environment
python -m venv venv
# (Activate it: venv\Scripts\activate on Windows or source venv/bin/activate on Mac)

# 4. Install Flask on their machine so the app runs for them too
pip install Flask
